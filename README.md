## Как установить pnpm (для Windows / Linux / macOS)

Откройте терминал (PowerShell, CMD, Terminal, zsh - любой) и выполните:

```bash
corepack enable
corepack prepare pnpm@latest --activate
```

## Как запустить проект

```bash
# 1. Установить зависимости
pnpm install
# 2. Запустить разработку
pnpm dev
```
